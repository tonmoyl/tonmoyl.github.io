  • Smart Rocket
    public web
    code github
    App Academy MAY 2018 - SEPTEMBER 2018
    New York, NY
    Smart Rocket is a simple Javascript demonstration which handles finding a path from a source and a target. The concept is for the rockets to break after hitting a barrier and fire rockets traveling in the opposite direction before collision. Once a rocket finally reaches its target, the Smart Rocket is formed. It knows the direction to get to the target and travels the path to reach the target.

    This project represents one of my first personal projects as a developer. Despite the simplicity of the demonstration, there were some interesting concepts used to make this project run smoothly. I used a tree to create a history of the vector directions that the rockets have taken. When the colliding rocket finally hits the target, a Smart Rocket is created. The colliding rocket’s direction of travel is saved in each parent node, thus resulting in a list of vectors the rockets took to reach its target.

    There are several ways to improve Smart Rocket. Each rocket after collision creates random sub-rockets traveling in random directions. It would improve efficiency if we reduce the randomness and make it travel in slight deviations of a known path. Another interesting concept is to create traits of its ancestors through some level of machine learning. There are a lot of interesting directions to take this project!

    App Academy MAY 2018 - SEPTEMBER 2018
    New York, NY
    DOMnoy is a project centered around creating a library that emulates jQuery. This tool is known to simplify a lot of common tasks in Javascript by providing functions that developers can use with small lines of code.

    The purpose of DOMnoy is to understand how popular libraries are created and the costs that might be associated with using these tools. Although jQuery makes the code much simpler, it was interesting to find out that most of the common features in the library can be utilized through Vanilla Javascript.

    DOMnoy currently supports several similar functions that jQuery has. Some of the features are searching through the HTML DOM for elements, manipulating HTML elements, adding/removing styling elements, appending strings, and setting attributes.
